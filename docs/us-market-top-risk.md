# 顶部风险预判

## 模块边界

本模块用于定期计算顶部风险状态，不直接给出买卖指令。当前正式页面是 `/risk`，
数据由 `backend/packages/public_app/market_top_risk.py` 每日生成并写入
`market_top_risk_snapshots`。

当前核心逻辑只看两层：

| 层级 | 含义 |
| --- | --- |
| 结构脆弱 | 宽度变窄、主线退潮或更宽的一侧跑输核心资产 |
| 价格转弱 | 市场自身出现回撤、跌破均线或均线下弯 |

`NFCI`、`ANFCI`、`BAA10Y`、实际利率、美元等宏观/信用压力指标不再进入正式评分。
它们太慢、太平滑，后续只适合作为研究背景。

## 正式状态机

结构和价格都每日按最新可用收盘价重算，但状态切换使用防噪声规则：

- 结构确认：最近 3 个交易日中至少 2 天结构分数 `>= 0.70`。
- 价格确认：最近 3 个交易日中至少 2 天价格转弱分数 `>= 0.50`。
- 顶部风险：过去 20 个交易日内出现过结构确认，并且当前价格确认。
- 破位确认：价格确认在最近 10 个交易日中至少出现 7 天。

状态解释：

| 状态 | 条件 | 含义 |
| --- | --- | --- |
| 健康上涨 | 结构不脆弱，价格不转弱 | 趋势正常 |
| 拥挤上涨 | 结构确认，价格未确认 | 风险变脆，但不能直接说见顶 |
| 普通回撤 | 价格确认，结构没有近期确认 | 可能只是波动或独立回撤 |
| 顶部风险 | 结构近期确认后价格确认 | 真正值得警惕 |
| 破位确认 | 价格转弱持续约 2 周 | 风险进入确认阶段 |

数据库兼容旧字段：

- `breadth_weakness_score` 现在表示综合结构分数。
- `breakage_score` 现在表示综合价格转弱分数。
- `warning_active` 表示结构确认。
- `confirmation_active` 表示价格确认。
- 详细状态矩阵写入 `metrics.markets`。

## 指标口径

### 美国结构

| 指标 | 含义 |
| --- | --- |
| `RSP/SPY` 13 周弱化 | 标普等权跑输市值加权，宽度变窄 |
| `QQEW/QQQ` 13 周弱化 | 纳指等权跑输 QQQ，科技内部集中 |
| `SOXX/QQQ` 13 周弱化 | 半导体主线跑输科技宽基 |
| `XLY/XLP` 13 周弱化 | 可选消费跑输必需消费，风险偏好转防御 |
| `IWM/SPY` 13 周弱化 | 小盘跑输大盘，风险承受能力下降 |

### 中国结构实验

中国侧先不使用 `588200/沪深300`，因为这更像高弹性主题相对宽基，容易失真。
当前只做内部宽度实验：

| 指标 | 含义 |
| --- | --- |
| `588120.SS/588000.SS` | 科创100ETF 是否跑输科创50ETF |
| `159915.SZ/159949.SZ` | 创业板ETF 是否跑输创业板50ETF |

这两个指标不是严格等权/市值加权口径，不单独作为买卖判断；它们只用于观察中国成长股
是否从更宽扩散收缩到更窄核心。

### 价格转弱

正式页面目前跟踪两条价格线：

| 市场 | 价格代理 |
| --- | --- |
| 美国半导体 | `SOXX` |
| 中国科创 | `588200.SH` |

价格转弱分数由四个条件等权组成：

- 距离 20 日高点回撤达到约 `-5%`。
- 收盘跌破 20 日均线。
- 收盘跌破 50 日均线。
- 20 日均线相对 5 个交易日前转负。

## 数据来源

| 数据 | 来源 | 用途 |
| --- | --- | --- |
| `NDX`、`SPY`、`RSP`、`QQQ`、`QQEW`、`SOXX`、`XLY`、`XLP`、`IWM` | Nasdaq 公共历史接口 | 美股结构与 SOXX 价格 |
| `588200.SS` | Yahoo Finance A 股代码 | 中国科创价格代理 |
| `588120.SS`、`588000.SS` | Yahoo Finance A 股代码 | 科创100/科创50 实验宽度 |
| `159915.SZ`、`159949.SZ` | Yahoo Finance A 股代码 | 创业板/创业板50 实验宽度 |

每个交易日收盘后同步任务都会尝试抓取最新数据。不同市场最新日期不强行对齐；
同一相对指标只使用同市场内部的最新可用收盘价。

## 页面展示

`/risk` 页面展示：

- 顶部风险总状态。
- `SOXX` 与 `588200.SH` 的状态矩阵。
- 结构分数、价格转弱分数和触发明细。
- 最近日度历史柱状图。

页面最新快照超过 5 天会提示数据陈旧。

## 调度建议

正式 worker 每天运行一次即可。推荐在中美主要市场收盘后运行。提醒逻辑后续应只在确认状态
变化时触发：

- `健康上涨 -> 拥挤上涨`：低优先级。
- `拥挤上涨 -> 顶部风险`：高优先级。
- `顶部风险 -> 破位确认`：高优先级。
- `顶部风险 -> 修复`：中优先级。


- `data/runtime/market_top_risk/report/market-top-risk-report.json`

不纳入自动指标。
